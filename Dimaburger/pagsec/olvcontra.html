Ingrese sus datos

En caso de que se le haya olvidado el nombre de usuario o la constraseña. Ingrese tu correo electronico para que podamos pasarte un codigo el cual nos permitira enviarle un link con el cual podra restaurar el nombre de Usuario y la contraseña.

¿Tenes problemas para registrarte?

En caso de que tengas problemas para registrarte puede ser por dos razones. La primera es que posees un correo que ya este asociado a alguna cuenta de Dimaburger, en cuyo caso te sugerimos que presiones el botón que dice: “Haz click para logearte”. Si el problema persiste te dejamos un link con el cual podrás acceder a un apartado de quejas y atención al cliente para que puedas solicitar ayuda.
